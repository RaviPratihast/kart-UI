Button

The Button component lets you add standard HTML buttons to your project. It allow users to take actions, and make choices, with a single tap.

Default Button

This is the default html tag button without any style. 

              
                
                <button>Default</button>

Standard Button

Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app. The 'btn' class is used for basic styling of button and classes 'btn-primary', 'btn-success', 'btn-danger' and 'btn-warning' for colors. 

              
                
                <button class="btn btn-primary">Primary</button>  
                <button class="btn btn-success">Success</button>  
                <button class="btn btn-danger">Danger</button>  
                <button class="btn btn-warning">Warning</button>

Icon Button

Icon buttons allow users to take actions, and make choices, with a single tap. Simply these are buttons with icons such as home notifications. The 'btn', 'btn-icon' , 'btn-menu' and 'btn-trash' class is used for basic styling and introducing icons to button. 

              
                
                <button class="btn btn-icon"><span class="material-symbols-outlined">House</span></button>  
                <button class="btn btn-icon"><span class="material-symbols-outlined">share</span></button>  
                <button class="btn btn-menu"><span class="material-symbols-outlined">folder</span></button>  
                <button class="btn btn-trash"><span class="material-symbols-outlined">delete</span></button>

Link Button

Link Buttons create a hyperlink-style button on the Web page. The Link Button control has the same appearance as a HyperLink control, but has the same functionality as a Button control. Here we have use the class 'btn btn-link' for basic button styling. 

              
                
                <button class="btn btn-icon"><a href="#">Link</a></button>

Floating Action Button

A floating action button (FAB) performs the primary, or most common, action on a screen. It appears in front of all screen content, typically as a circular shape with an icon in its center. Here we have used classes 'btn btn-float' for basic styling. 

              
                
                <button class="btn btn-floating"><span class="material-symbols-outlined">add_circle</span></button>
