Input

Input fields are an essential user interface design element, providing users with the means to enter non-standardized responses. Input component has a range of options and supports several text formats including numbers.

Textbox

This is an general input component i.e., textbox. Textarea tag is is used along with class 'input-box' for basic styling. Also Input tag can be used to create an text box with type specified as text. 

              
                <textarea name="text" class="text-input" cols="30" rows="10"></textarea>

Input with validation

This is an general input component with validation provided. It can be of different natures like success or invalid.

Invalid Email Address

              
                <input type="email" class="input-mail">
                <p class="email-validation-text">Invalid Email Address</p>
