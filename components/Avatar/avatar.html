Avatar

The Avatar component is used to represent a user, and displays the profile picture, initials or fallback icon.

Profile Avatar

Profile Avatars can be created using the class 'avatar', inside the div tag and also use the classes 'avatar-image avatar-circle' inside the img tag. 

              
                
                <div class="avatar-container">
                <img src="/images/avatar-profile.jpg" alt="avatar-profile"
                class="avatar-profile"/></div>

Square Avatar

Rounded Square Avatars can be created using the class 'avatar', inside the div tag and also use the classes 'avatar-image avatar-square' inside the img tag. 

              
                
                <div class="avatar-container">
                <img src="/images/avatar-profile.jpg" alt="avatar-profile"
                class="avatar-square"/></div>

Sized Avatar

Size of the avatars can be changed by adding avatar-small, avatar-medium or avatar-large class to the existing class in the img tag as shown in code snippet. 

              
                
                <div class="avatar-container">
                <img src="/images/avatar-profile.jpg" alt="avatar-profile"
                class="avatar-small"/></div>

                <div class="avatar-container">
                <img src="/images/avatar-profile.jpg" alt="avatar-profile"
                class="avatar-medium"/></div>

                <div class="avatar-container">
                <img src="/images/avatar-profile.jpg" alt="avatar-profile"
                class="avatar-large"/></div>
